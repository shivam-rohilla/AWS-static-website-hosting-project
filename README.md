\# AWS Static Website Hosting



\## Overview

This project demonstrates hosting a static website on AWS using Amazon S3.



\## Architecture

The website is hosted in an S3 bucket with public read access enabled

using bucket policies.



\## AWS Services Used

\- Amazon S3

\- IAM



\## Features

\- Static website hosting

\- Public access via S3 website endpoint

\- Simple and cost-effective architecture



\## What I Learned

\- Hosting static content on AWS

\- S3 bucket policies and permissions

\- Cloud-based web hosting fundamentals

Resources were deleted to avoid unnecessary cloud costs.
